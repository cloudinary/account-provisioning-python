# QueryParamSortOrder

Control the order of returned users. **Possible values**: `desc` (default), `asc`.


## Example Usage

```python
from cloudinary_account_provisioning.models import QueryParamSortOrder

value = QueryParamSortOrder.DESC
```


## Values

| Name   | Value  |
| ------ | ------ |
| `DESC` | desc   |
| `ASC`  | asc    |
