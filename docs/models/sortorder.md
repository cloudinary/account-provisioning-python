# SortOrder

Control the order of returned keys. **Possible values**: `desc` (default), `asc`.


## Example Usage

```python
from cloudinary_account_provisioning.models import SortOrder

value = SortOrder.DESC
```


## Values

| Name   | Value  |
| ------ | ------ |
| `DESC` | desc   |
| `ASC`  | asc    |
